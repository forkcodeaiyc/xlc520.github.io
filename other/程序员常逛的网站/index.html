# 程序员常逛的网站

# 一、在线教程

首先列出一些在线教程网站，这些在线教程网站通常都比较适合入门，可以作为开发学习路上的第一个阶梯，也可以作为工作中的在线文档。

# 1、http://how2j.cn (opens new window)

  • 地 址：https://how2j.cn/ (opens new window)

  • 简 介：一个 Java 全栈开发教程网站，内容全面，简洁易懂，非常适合入门。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、w3cschool

  • 地 址：https://www.w3cschool.cn/ (opens new window)

  • 简 介：前端和脚本语言为主的在线教程网站，前端的内容非常不错。

  • 推荐指数：⭐⭐⭐⭐⭐

# 3、菜鸟教程

  • 地 址：https://www.runoob.com/ (opens new window)

  • 简 介：和 w3cschool 类似的在线教程网站，前端学习看这两个网站就够了。

  • 推荐指数：⭐⭐⭐⭐⭐

# 4、易百教程

  • 地 址：https://www.yiibai.com/ (opens new window)

  • 简 介：内容比较全面的在线教程网站。

  • 推荐指数：⭐⭐⭐

# 9、并发编程网

  • 地 址：http://ifeve.com/ (opens new window)

  • 简 介：比较不错的技术网站，以Java为主，关注并发、NIO、JVM、框架等方面的内容。

  • 推荐指数：⭐⭐

# 10、http://jenkov.com (opens new window)

  • 地 址：http://tutorials.jenkov.com/ (opens new window)

  • 简 介：一个国外的技术网站，提供了非常不错的Java教程。

  • 推荐指数：⭐⭐⭐

# 11、http://baeldung.com (opens new window)

  • 地 址：https://www.baeldung.com/ (opens new window)

  • 简 介：也是一个国外的技术网站，提供了非常不错的Java教程。

  • 推荐指数：⭐⭐⭐

# 二、视频教程

上面的在线教程网站以文字性内容为主，同样收藏了一些以视频为主的一些网站，相比较而言，更容易上手，当然，也更加耗时间。

# 1、B站

  • 地 址：https://www.bilibili.com/ (opens new window)

  • 简 介：没错，bilibili也可以用来学习，除了番剧、鬼畜之外，b站同样有非常多的高质量的编程教学视频，搜索就行了。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、慕课网

  • 地 址：https://www.imooc.com/ (opens new window)

  • 简 介：比较不错的编程视频教学网站，可以找到比较体系的东西，当然，有些内容要花钱就是了。

  • 推荐指数：⭐⭐⭐⭐

# 3、中国大学MOOC

  • 地 址：https://www.icourse163.org/ (opens new window)

  • 简 介：有很多名校的课程，当然不止CS。

  • 推荐指数：⭐⭐⭐⭐

# 4、网易云课堂

  • 地 址：https://study.163.com (opens new window)

  • 简 介：有一些计算机编程方面的视频教程。

  • 推荐指数：⭐⭐⭐

# 5、实验楼

  • 地 址：https://www.lanqiao.cn/courses/ (opens new window)

  • 简 介：IT技术课程网站。

  • 推荐指数：⭐⭐

# 6、我要自学网

  • 地 址：（敏感词，自行百度）

  • 简 介：比较多的自学教程，有程序设计内容的分类。

  • 推荐指数：⭐⭐

# 7、大学生自学网

  • 地 址：http://v.dxsbb.com/jisuanji/ (opens new window)

  • 简 介：有一些计算机软件的大学课程。

  • 推荐指数：⭐⭐

# 8、极客学院

  • 地 址：https://www.jikexueyuan.com/ (opens new window)

  • 简 介：IT编程课程网站，付费内容居多。

  • 推荐指数：⭐⭐

# 四、官网

毫无疑问，一项技术，最权威的文档一定是它的官方文档，所有的教程、书籍都是直接或者间接在官方相关文档的基础上完成。所以，编程学习，后面最好还是要看官网的。当然，官网大部分都是英文——其实配合一些翻译插件，没有想象中的那么难。

# 1、Java

  • 地 址：https://www.java.com/zh_CN/ (opens new window) 、https://docs.oracle.com/en/java/javase/index.html (opens new window)

  • 简 介：这个不需要多介绍，Java官网和Java官方API。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、Spring

  • 地 址：https://spring.io/ (opens new window)

  • 简 介：毫无疑问Spring已经成为Java EE事实上的标准。

  • 推荐指数：⭐⭐⭐⭐⭐

# 3、MySQL

  • 地 址：https://www.mysql.com/ (opens new window)

  • 简 介：MySQL官网

  • 推荐指数：⭐⭐⭐⭐

# 4、MyBatis

  • 地 址：https://mybatis.org/mybatis-3/zh/index.html (opens new window)

  • 简 介：国内最流行的Java持久层框架，而且幸运的是，MyBatis文档是有中文版的。

  • 推荐指数：⭐⭐⭐⭐

# 5、Vue.js

  • 地 址：https://cn.vuejs.org/ (opens new window)

  • 简 介：国内最流行的前端SOP框架，Vue.js的文档是中文的。

  • 推荐指数：⭐⭐⭐⭐

# 6、Linux

  • 地 址：https://www.linux.org/ (opens new window)

  • 简 介：后端程序员必会的Linux.

  • 推荐指数：⭐⭐⭐⭐

# 7、Git

  • 地 址：https://git-scm.com/ (opens new window)

  • 简 介：版本管理推荐用Git，同样有中文版本的文档。

  • 推荐指数：⭐⭐⭐⭐

# 8、Dubbo

  • 地 址：http://dubbo.apache.org/zh-cn/ (opens new window)

  • 简 介：一款高性能的Java RPC框架，国内用的还是比较广泛，源于阿里，中文文档是有的。

  • 推荐指数：⭐⭐⭐⭐

# 9、Redis

  • 地 址：https://redis.io/ (opens new window)

  • 简 介：最流行的No SQL数据库。

  • 推荐指数：⭐⭐⭐⭐

# 五、国内博客社区

了解技术动态、解决各种问题都会用到技术博客。有些优质博主同样创作了一些比较优秀的技术教程。

# 1、CSDN

  • 地 址：https://blog.csdn.net/kexuanxiu1163 (opens new window)

  • 简 介：最大的中文技术博客社区，内容最多，SEO做的也不错。

  • 推荐指数：⭐⭐⭐

# 2、博客园

  • 地 址：https://www.cnblogs.com/ (opens new window)

  • 简 介：老牌中文技术博客社区，博客质量比较高，商业化气息也比较淡，之前广为诟病的UI风格也在今年进行了改进，自定义主题也让能让博客百花争艳。

  • 推荐指数：⭐⭐⭐⭐⭐

# 3、掘金

  • 地 址：https://juejin.im/user/4230576474426685 (opens new window)

  • 简 介：UI做的最漂亮的中文技术博客社区，内容以前端和面试居多。

  • 推荐指数：⭐⭐⭐⭐

# 4、简书

  • 地 址：https://www.jianshu.com/ (opens new window)

  • 简 介：并不是纯粹的技术博客社区，很多伤春悲秋的东西。

  • 推荐指数：⭐⭐⭐

# 5、思否

  • 地 址：https://segmentfault.com/ (opens new window)

  • 简 介：包含博客、问答的技术社区。

  • 推荐指数：⭐⭐⭐

# 6、开源中国

  • 地 址：https://www.oschina.net (opens new window)

  • 简 介：一个技术博客社区。

  • 推荐指数：⭐⭐⭐

# 7、http://51CTO.COM (opens new window)

# 六、国外技术博客社区

上面介绍了一些国内的博客社区，事实上，放眼看世界也很重要，毕竟我们用的技术大部分都来自国外。当然，其实更加建议想办法访问谷歌，个人觉得一个谷歌足矣。

# 1、Stack Overflow

  • 地 址: https://stackoverflow.com/ (opens new window)

  • 简 介：全球最活跃的程序员技术问答交流社区，有人说程序员的所有问题都能在上面找到答案。

  • 地 址: https://dev.to/ (opens new window)

# 3、DZone

  • 地 址: https://dzone.com/ (opens new window)

  • 简 介：http://DZone.com (opens new window)是世界上最大的在线社区之一。

  • 推荐指数：⭐⭐⭐

# 4、Bytes

  • 地 址: https://bytes.com/ (opens new window)

  • 简 介：一个面向开发人员和IT专业人员的交流社区。

  • 推荐指数：⭐⭐⭐

# 5、Google Developers

  • 地 址: https://developers.google.com/ (opens new window)

  • 简 介：google开发社区。

  • 推荐指数：⭐⭐⭐
